# 🎮 Tic Tac Toe Game

A beautifully designed, modern Tic Tac Toe game built with HTML, CSS, and JavaScript. Play against a friend with an intuitive interface, smooth animations, and responsive design.
## ✨ Features

- 🎨 **Modern UI/UX**: Beautiful gradient background with smooth animations
- 📱 **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- 🎯 **Interactive Gameplay**: Real-time player turn indicator
- 🏆 **Winner Detection**: Automatically detects wins and draws
- ⚡ **Smooth Animations**: Pop-in, scale, and slide animations for engaging experience
- 🔄 **Easy Reset**: One-click reset or play again functionality
- ♿ **Accessible**: Keyboard and mouse friendly controls

## 🎮 How to Play

1. **Start the Game**: Open `index.html` in your web browser
2. **Players**: Player X always goes first
3. **Make a Move**: Click on any empty cell to place your mark
4. **Win**: Get three of your marks in a row (horizontally, vertically, or diagonally)
5. **Draw**: If all cells are filled with no winner, it's a draw
6. **Reset**: Click "Reset Game" to start over anytime
7. **Play Again**: After a game ends, click "Play Again" to start fresh

## 📦 Installation

### Option 1: Direct File Opening
```bash
1. Clone or download the repository
2. Navigate to the Tic_Tac_Toe_Game folder
3. Double-click index.html to open in your browser
```

### Option 2: Using a Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
http-server
```

Then open `http://localhost:8000` in your browser.

## 🎯 Game Rules

- The game is played on a 3x3 grid
- Two players take turns: X and O
- Player X goes first
- Players mark their symbol on empty cells
- The first player to get 3 marks in a row wins
- If all 9 cells are filled with no winner, the game is a draw
